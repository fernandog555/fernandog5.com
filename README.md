# Fernando Flores — Portfolio

Personal portfolio built with React, Vite, and Tailwind. It highlights recent work and offers quick navigation to projects and contact.

## Tech
- React + Vite
- Tailwind CSS
- Dark/light theme toggle (system-aware)

## Getting Started
```bash
npm install
npm run dev
```
Open `http://localhost:5173` in your browser.

## Scripts
- `npm run dev` — start the local dev server
- `npm run build` — production build
- `npm run preview` — preview the production build
- `npm run lint` — run eslint
